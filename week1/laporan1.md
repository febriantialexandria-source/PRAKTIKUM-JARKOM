# Laporan Praktikum Jarkom

## Tujuan Praktikum
Menginstall dan mempelajari software wireshark

## Langkah percobaan
1. Download wireshark.org dan install aplikasi Wireshark 4.6.4 x64.
2. Setelah itu, buka Wireshark lalu pilih Wifi pada bagian network (jika menggunakan Wifi).
3. Jika sudah terinstall, buka Wireshark lalu pilih Wifi pada bagian network (jika menggunakan Wifi).
4. Jika ingin menampilkan paket dengan protokol tcp saja, klik display filter bar, lalu ketik tcp dan tekan enter.
5. Setelah itu saat Wireshark sedang melakukan capture, buka browser dan masukkan URL berikut:
http://gaia.cs.umass.edu/wireshark-labs/INTRO-wireshark-file1.html lalu buka halaman tersebut.
6. Setelah halaman terbuka, kembali ke Wireshark dan hentikan capture dengan menekan tombol stop (ikon merah).
7. Setelah itu, pada bagian filter, ketik http untuk menampilkan paket dengan protokol HTTP.
