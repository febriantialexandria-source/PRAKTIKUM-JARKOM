# Laporan Praktikum Jarkom 2 (2_1 Basic HTTP GET/response interaction)

## Tujuan Praktikum
Agar mahasiswa dapat mempelajari dan menganalisis cara kerja protokol HTTP dengan menggunakan Wireshark.

## Langkah percobaan
1. Buka aplikasi Wireshark.
2. Lalu klik jaringan WiFi.
3. Di kotak filter atas, ketik http supaya yang muncul cuma HTTP.
4. Setelah itu klik tombol start untuk mulai menangkap data.
5. Lalu buka browser dan ketik: http://gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file1.html
6. Setelah halaman terbuka, balik lagi ke Wireshark.
7. Terakhir, klik tombol stop untuk berhenti.
