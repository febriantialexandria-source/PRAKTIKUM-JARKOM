# Laporan Praktikum Jarkom 2 (2_1 Basic HTTP GET/response interaction)

## Tujuan Praktikum
Mempelajari dan menganalisis cara kerja protokol HTTP dengan menggunakan Wireshark.

## Langkah percobaan
1. Buka aplikasi Wireshark.
2. Kemudian klik jaringan WiFi.
3. Di kotak filter atas, ketik http supaya yang muncul cuma HTTP.
4. Lalu klik tombol start untuk mulai menangkap data.
5. Setelah itu buka browser dan ketik: http://gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file1.html
6. Setelah halaman terbuka, kembali lagi ke Wireshark.
7. Terakhir, klik tombol stop untuk berhenti.
